#  Интернет-магазин компьютерных комплектующих
Веб-приложение для продажи компьютерных комплектующих с возможностью фильтрации товаров, добавления в корзину и управления заказами. Реализовано на легковесном фреймворке Python Bottle с использованием SQLite для хранения данных.

##  О проекте

Интернет-магазин специализируется на продаже компьютерных комплектующих. Проект создан с целью изучения и практического применения веб-технологий, включая работу с базой данных, создание REST API, разработку клиент-серверного взаимодействия и создание адаптивного пользовательского интерфейса.

## 🛠️ Технологический стек

**Backend:**
- Python 3.14
- Bottle Framework (0.13+)
- SQLite 3

**Frontend:**
- HTML5
- CSS3 (адаптивный дизайн)
- JavaScript (ES6+)
- Fetch API для AJAX-запросов

##  Функциональность

### Основные возможности:
-  **Каталог товаров** - просмотр всех доступных товаров
-  **Фильтрация по категориям** - материнские платы, процессоры, память, охлаждение, расходники, видеокарты
-  **Фильтрация по цене** - настраиваемый диапазон цен
-  **Фильтрация по наличию** - только товары в наличии
-  **Корзина покупок** - добавление и управление товарами
-  **Актуальное наличие** - автоматическое обновление количества на складе

### Интерфейс:
-  **Адаптивный дизайн** - корректное отображение на всех устройствах
-  **Модальные окна** - для корзины и фильтров
-  **Уведомления** - информативные сообщения о действиях
-  **Единообразные изображения** - автоматическое приведение к одному размеру

## 🗄️ База данных

### Таблица `products`
| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER PRIMARY KEY AUTOINCREMENT | Уникальный идентификатор |
| name | TEXT NOT NULL | Название товара |
| description | TEXT | Описание товара |
| price | REAL NOT NULL | Цена в рублях |
| stock_quantity | INTEGER DEFAULT 0 | Количество на складе |
| category | TEXT | Категория товара |
| image_url | TEXT | Путь к изображению |
| created_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | Дата добавления |

### Таблица `cart`
| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER PRIMARY KEY AUTOINCREMENT | Уникальный идентификатор |
| user_id | INTEGER NOT NULL | ID пользователя |
| product_id | INTEGER NOT NULL | ID товара (foreign key) |
| quantity | INTEGER DEFAULT 1 | Количество |
| added_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | Время добавления |
| UNIQUE(user_id, product_id) | - | Уникальная пара пользователь-товар |

## 🔧 Установка и запуск

### Предварительные требования
- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Пошаговая установка

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/your-username/computer-shop.git
cd computer-shop
```

2. **Установите зависимости**
```bash
pip install bottle
```
Или используя requirements.txt:
```bash
pip install -r requirements.txt
```

3. **Запустите приложение**
```bash
python app.py
```

4. **Откройте в браузере**
```
http://localhost:8080
```

## 🌐 API Endpoints

| Метод | URL | Описание | Параметры |
|-------|-----|----------|-----------|
| GET | `/` | Главная страница | - |
| GET | `/api/products` | Получить все товары | - |
| GET | `/api/products/filter` | Фильтрация товаров | `categories` (через запятую), `minPrice`, `maxPrice`, `inStock` |
| GET | `/api/products/<id>` | Получить конкретный товар | `id` - идентификатор товара |
| GET | `/api/cart` | Получить корзину | - |
| POST | `/api/cart/add` | Добавить в корзину | `product_id`, `quantity` (JSON) |
| POST | `/api/cart/clear` | Очистить корзину | - |
| GET | `/static/<filename>` | Статические файлы | `filename` - путь к файлу |
